Free Stream Ciphers

Эта программа реализует шифрование с использованием потоковых шифров.
Для начала вам необходимо запустить программу. И вписать одну из четырех команд.
Их список вы можете посмотреть здесь, а также ввести команду "help".

Команды:
1. encode - Код вашего текста.
2. decode - Расшифруйте ваш текстовый файл.
3. help - Справочник.
4. exit - Выход из программы.

Программа также может принимать аргументы. Для того, чтобы зашифровать текст, написанный вне программы,
создайте файл «.txt» и перетащите его на «RSC.exe».
Программа автоматически рассчитывает путь, и все, что остается сделать, это ввести пароль.

Внимание, программа не принимает большой текст с переносами строк.
Для того чтобы программа правильно зашифровала текст, необходимо ввести текст без переноса строки.

Чтобы расшифровать текст таким же образом, вам нужно переместить текстовый файл в «RSC.exe».
Далее выберите команду "decode" и введите пароль. Программа сама рассчитает путь до файла.

Рекомендуется хранить пароли для учетных записей, банковских карт и ценную информацию в зашифрованных файлах, это повысит вашу безопасность.
